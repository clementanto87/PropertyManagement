# Property Management Frontend

Mobile-first responsive React application for property management.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy environment file:
```bash
cp .env.example .env
```

3. Update `.env` with your backend API URL (default: `http://localhost:3000/api`)

4. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Features

- Mobile-first responsive design
- Dashboard with key metrics
- Property, Unit, Tenant, Lease management
- Work Orders and Maintenance tracking
- Vendor management
- Expense tracking
- Bottom navigation for mobile

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Lucide React (icons)
